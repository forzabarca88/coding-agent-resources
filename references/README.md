## Contributing Reference Materials

When adding reference materials:

1. **Organize by topic**: Create subdirectories for related materials
2. **Use clear naming**: Use descriptive filenames (e.g., `api-authentication.md`)
3. **Include examples**: Provide practical examples where possible
4. **Cross-reference**: Link to related materials in other directories
5. **Keep updated**: Ensure reference materials stay current with code changes
